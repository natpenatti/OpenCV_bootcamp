This repository contains my Jupyter Notebooks, exercises, and personal notes from the **OpenCV Bootcamp** offered by [\[OpenCV University\]](https://opencv.org/university/)([OpenCV Bootcamp](https://opencv.org/university/free-opencv-course/?utm_source=opcvu&utm_medium=menu&utm_campaign=obc)). 

The course provides a fast-paced, hands-on introduction to Computer Vision and image processing using Python and the OpenCV library. Currently, I am actively working through the modules.

## 📂 Repository Structure

As I progress through the course, the project is being structured around the main Jupyter Notebooks:

* `notebooks/`
  * `01_Getting_started_with_images.ipynb`
  * `02_Basic_Image_Manipulations_Crop_Resize_Flip_and_Modify_Pixels.ipynb`
  * `03_Annotating_images.ipynb`
  * `04_Basic_Image_ENhancement_mathematical_Operations.ipynb`
* `assets/` - Contains the sample images, videos, and other assets used across the notebooks.
